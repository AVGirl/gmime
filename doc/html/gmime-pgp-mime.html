pgp_mime_init ()

void        pgp_mime_init                   (const gchar *path,
                                             PgpType type,
                                             PgpPasswdFunc callback,
                                             gpointer data);

Initializes pgp-utils (same as calling pgp_init with the same args)

path : path to pgp program
type : type of pgp program
callback : the get-password callback
data : user data

pgp_mime_part_is_rfc2015_signed ()

gboolean    pgp_mime_part_is_rfc2015_signed (GMimePart *mime_part);

mime_part : MIME Part
Returns :TRUE if it is an rfc2015 multipart/signed.

pgp_mime_part_is_rfc2015_encrypted ()

gboolean    pgp_mime_part_is_rfc2015_encrypted
                                            (GMimePart *mime_part);

mime_part : MIME Part
Returns :TRUE if it is an rfc2015 multipart/encrypted.

pgp_mime_part_sign ()

void        pgp_mime_part_sign              (GMimePart **mime_part,
                                             const gchar *userid,
                                             PgpHashType hash,
                                             GMimeException *ex);

Constructs a PGP/MIME multipart in compliance with rfc2015 and replaces part with the generated multipart/signed. On failure, ex will be set and part will remain untouched.

mime_part : a MIME part that will be replaced by a pgp signed part
userid : userid to sign with
hash : one of PGP_HASH_TYPE_MD5 or PGP_HASH_TYPE_SHA1
ex : exception which will be set if there are any errors.

pgp_mime_part_verify_signature ()

gboolean    pgp_mime_part_verify_signature  (GMimePart *mime_part,
                                             GMimeException *ex);

mime_part : a multipart/signed MIME Part
ex : exception
Returns :TRUE if the signature is valid otherwise returns FALSE. Note: you may want to check the exception if it fails as there may be useful information to give to the user; example: verification may have failed merely because the user doesn't have the sender's key on her system.

pgp_mime_part_encrypt ()

void        pgp_mime_part_encrypt           (GMimePart **mime_part,
                                             const GPtrArray *recipients,
                                             GMimeException *ex);

Constructs a PGP/MIME multipart in compliance with rfc2015 and replaces part with the generated multipart/signed. On failure, ex will be set and part will remain untouched.

mime_part : a MIME part that will be replaced by a pgp encrypted part
recipients : list of recipient PGP Key IDs
ex : exception which will be set if there are any errors.

pgp_mime_part_decrypt ()

GMimePart*  pgp_mime_part_decrypt           (GMimePart *mime_part,
                                             GMimeException *ex);

mime_part : a multipart/encrypted MIME Part
ex : exception
Returns :the decrypted MIME Part on success or NULL on fail.
