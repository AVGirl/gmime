struct GMimeDataWrapper

struct GMimeDataWrapper {
	GObject parent_object;
	
	GMimePartEncodingType encoding;
	GMimeStream *stream;
};

stream :

encoding :

stream's encoding

Returns :

a data wrapper around stream. Since the wrapper owns it's own reference on the stream, caller is responsible for unrefing it's own copy.
