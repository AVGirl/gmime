g_mime_header_set ()

void        g_mime_header_set               (GMimeHeader *header,
                                             const gchar *name,
                                             const gchar *value);

Set the value of the specified header. If value is NULL and the header, name, had not been previously set, a space will be set aside for it (useful for setting the order of headers before values can be obtained for them) otherwise the header will be removed.

header : header object
name : header name
value : header value (or NULL to remove header)
