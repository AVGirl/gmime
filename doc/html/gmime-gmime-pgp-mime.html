g_mime_pgp_mime_is_rfc2015_signed ()

gboolean    g_mime_pgp_mime_is_rfc2015_signed
                                            (GMimePart *mime_part);

Examines the part to decide whether or not it is an rfc2015 signed part.

mime_part : a multipart/signed mime part
Returns :TRUE if mime_part is an rfc2015 signed part or FALSE otherwise.

g_mime_pgp_mime_is_rfc2015_encrypted ()

gboolean    g_mime_pgp_mime_is_rfc2015_encrypted
                                            (GMimePart *mime_part);

Examines the part to decide whether or not it is an rfc2015 encrypted part.

mime_part : a multipart/encrypted mime part
Returns :TRUE if mime_part is an rfc2015 encrypted part or FALSE otherwise.

g_mime_pgp_mime_part_sign ()

void        g_mime_pgp_mime_part_sign       (GMimePgpContext *context,
                                             GMimePart **mime_part,
                                             const char *userid,
                                             GMimeCipherHash hash,
                                             GMimeException *ex);

Constructs a PGP/MIME multipart in compliance with rfc2015 and replaces mime_part with the generated multipart/signed. On failure, ex will be set and part will remain untouched.

context : PGP Context
mime_part : a MIME part that will be replaced by a pgp signed part
userid : userid to sign with
hash : one of GMIME_CIPHER_HASH_MD5 or GMIME_CIPHER_HASH_SHA1
ex : exception which will be set if there are any errors.

GMimeCipherValidity* g_mime_pgp_mime_part_verify
                                            (GMimePgpContext *context,
                                             GMimePart *mime_part,
                                             GMimeException *ex);

Attempts to verify the signature of the multipart/signed mime part.

context : PGP Context
mime_part : a multipart/signed MIME Part
ex : exception
Returns :a GMimeCipherValidity on success or NULL on fail.

g_mime_pgp_mime_part_encrypt ()
